To edit this file on GitHub, click the pencil button in the upper right corner of the window. This will open the editor where you can make changes to the file.

After you have made changes to the file, scroll down to the "Commit Changes" section.

Enter a description of the changes you made, then click the "Commit Changes" button.

To create a pull request, click the "Pull Request" button in the upper right corner of the window.

Here you can enter the title and description of the request, and then click "Create Pull Request."

The repository creator will now receive a notification of your pull request and can decide whether to accept or reject it.
