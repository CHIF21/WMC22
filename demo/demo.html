Demo Repo for WMC22

Um diese Datei auf GitHub zu bearbeiten, klicken Sie auf den Stift-Button in der rechten oberen Ecke des Fensters. Dies öffnet den Editor, in dem Sie Änderungen an der Datei vornehmen können.

Nachdem Sie Änderungen an der Datei vorgenommen haben, scrollen Sie nach unten zum Abschnitt "Commit Changes".

Geben Sie eine Beschreibung der Änderungen ein, die Sie vorgenommen haben, und klicken Sie dann auf die Schaltfläche "Commit Changes".

Um eine Pull-Anfrage zu erstellen, klicken Sie auf die Schaltfläche "Pull Request" in der rechten oberen Ecke des Fensters.

Hier können Sie den Titel und die Beschreibung der Anfrage eingeben und dann auf "Create Pull Request" klicken.

Der Ersteller des Repositorys erhält nun eine Benachrichtigung über Ihre Pull-Anfrage und kann entscheiden, ob er sie akzeptieren oder ablehnen möchte.
